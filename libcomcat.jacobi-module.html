Function Details

jacobi(A, eps=1e-12)

source code 

This routine finds eigenvalues and -vectors for a symmetric matrix by a cyclic Jacobi rotation algorithm.

Parameters:
  • A - A The matrix (NumPy array) whos eigenvalues and -vectors are to be determined.
  • eps - The desired accuracy (zeroing limit).
Returns:
[E, V, s] Numpy array of eigenvalues and of eigenvectors, as well as number of sweeps used.

rotate(p, q, A, V)

source code 

This subroutine performs Jacobi rotation on a matrix A, eliminating A[pq].

Parameters:
  • p - indices of the element to be eliminated.
  • q - indices of the element to be eliminated.
  • A - The matrix to be diagonalised.
  • V - The matrix of eigenvectors.
Returns:
[A, V] Updated A and V matrices.
